Incremento 
======================

Vamos ver agora com mais calma o operação de soma que vimos de relance na aula anterior;

```javascript
  var x = 1
  var soma = x + 1
  
  document.write(soma);
```

Você consegue adivinhar o que vai aparecer na tela?

👵🏻 Claro não sou burra
--------------------------

Hehehe não mesmo. Mas existe uma forma ainda mais simples de escrever isso e conseguir o mesmo resultado:

```javascript
  var x = 1
  soma = ++x
  
  document.write(soma);
```

Quando você usa ++x o script entende que você quer somar 1 ao valor de x.  Na verdade você nem precisaria declarar a variavel soma:

```javascript
  var x = 1
  ++x
  
  document.write(x);
```

## Incrementos maiores que 1

Para usar o incremento com valores maiores que 1 asta usar a expressão **+=**:

```javascript
  var x = 1
  x += 10
  
  document.write(x);
```
