Aritmética
================================

Você sabe quais são as quatro operações da aritmética?

👵🏻 Adição, subtração, multiplicação e divisão,
---------------------------------------------------

Isso mesmo. Vamos ver cada uma delas agora.

## Adição

Como há vimos no javascript uma soma se faz usando o simbolo +

```javascript
  var x = 2
  var y = 3
  
  var resultado = x + y;
  
  document.write(resultado);
```

Será que você consegue mudar o código acima para fazer uma subtração? Faça uma tentativa antes de continuarmos.

## Subtração

Se você simplesmente trocou o simbolo + por - você acertou:

```javascript
  var x = 2
  var y = 3
  
  var resultado = x - y;
  
  document.write(resultado);
```

## Multiplição

Para fazer uma divisão usamos o símbolo *

```javascript
  var x = 10
  var y = 5
  
  var resultado = x * y;
  
  document.write(resultado);
```

## Divisão

Por fim a divisão é feita usando o símbolo /

```javascript
  var x = 10
  var y = 5
  
  var resultado = x / y;
  
  document.write(resultado);
```
