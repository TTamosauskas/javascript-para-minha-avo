Outputs
==========

Outputs são maneiras do computador enviar alguma informação de volta para nós. 

No script abaixo o output é a última linha, a que entrega a mensagem para a tela:

```javascript
var mensagem = "Olá Mundo!";

document.write(mensagem);
```

👵🏻 Existem outras formas de output?
------------------------------------

Existem sim. Experimente o script abaixo que simplesmente troca **document.write** por **alert**

```javascript
var mensagem = "Olá Mundo!";

alert(mensagem);
```

👵🏻 Gostei! Tem mais?

Tem sim, outra forma bastante comum do seu programa enviar um retorno para nós é o **console.log**. Experimente

```javascript
var mensagem = "Olá Mundo!";

console.log(mensagem);
```

👵🏻 Não apareceu nada

Isso porque as mensagens do console.log ficam meio escondidas, mas este é um recurso bastante usado pelos programadores.

O Exercício desta lição é exatamente encontrar elas.

Exercício
============

1. Abra o script acima novamente no seu navegador
2. Aperte a **tecla F12**.
2. Em e seguida clique na opção **Console**.

Assim você será capaz de ler as mensagens do console.log. 

Atualize a variavel mensagem e execute novamente a página para ver um novo console
