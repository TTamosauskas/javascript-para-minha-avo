
Função com parámetros
-----------------------

Outro recurso poderoso das funções é a capacidade de receber parametros.

```javascript
function soma(numero1, numero2) {
  var resultado = numero1 + numero2
  alert(resultado)
};

soma(1,2)
```

A função acima vai pegar os valores 1 e 2 e usar para fazer uma soma. 

Experimente repetir a última linha com dois paraetros diferentes e veja o que acontece.

============

Crie uma função chamada **multiplicacao()** que multiplique dois parametros que você enviar.
