
Função com parámetros
-----------------------

Outro recurso poderoso das funções é a capacidade de receber parametros.

👵🏻 O que são parametros?
-------------------------

São um tipo especial de variavel que são usadas dentro de uma função. Observe:

```javascript
function soma(parametro1, parametro2) {
  
  var resultado = parametro1 + parametro2
  
  alert(resultado)
};

soma(1,2)
```

A função acima vai pegar os parametros 1 e 2 e usar para fazer uma soma. 

Experimente repetir a última linha com dois paraetros diferentes e veja o que acontece.
