Comparando Variaveis
=====================

O If() pode ser usado para comparar também variaveis não numéricas.

Comparando Booleanos
---------------------

Como vimos nas primeiras aulas uma variavel booleana é aquela que é definida como true ou false, ou seja, verdadeira ou falso. Veja o Exemplo a seguir:

```javascript
var autorizacao = true
if (autorizacao)
{
alert("Autorização OK");
}
```

No exemplo acima a caixa de alerta só vai aparecer se a autorizacao existir. Ou seja, se for false ela não aparecerá.

Comparando Strings
---------------------

O comando if pode ser usado também para comparar strings, variaveis do tipo texto:

```javascript
var sexo = "F"
if (sexo == "F"){
alert("Feminino")
}
```
O sinal **==** na segunda linha significa que a condição if só será aceita se a variavel sexo for exatamente igual a M.
