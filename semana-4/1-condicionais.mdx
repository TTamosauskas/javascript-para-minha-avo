Condicionais
==============

Uma das coisas mais legais que um programa pode fazer é aprender a tomar decisões. 

👵🏻 Parece útil!
-----------------
E é.. mas para tomar decisões nosso programa precisará que passemos algumas orientações. 
 
Para fazer isso usando o **If**. Em inglês If significa **Se**, então ele fará algo **SE** nosso parametro for verdadeiro.

Este padrão, pode ser numérico por exemplo:

```javascript
var numero = 12

if (numero>10)
{
alert("Número maior que 10");
}
```
Para ficar bem claro. Se os parametros dentro dos parenteses **if(      )** forem verdadeiros, então o script executará o que está logo em seguida dentro das chaves **{           }**

Faça algum testes como script acima mudando o número. Note que ele apenas mostrará a mensagem se número for maior que 10.

Operadores comuns
----------------------

* **if (x)** x existe?
* **if (!x)** x não existe?
* **if (x == y)** x é igual a y?
* **if (x !== y)** x é diferente de y?
* **if (x > y)** x é maior que y?
* **if (x >= y)** é maior ou igual a y?
* **if (x < y)** x é menor que y?
* **if (x =< y)** x é menor ou igual a y?
