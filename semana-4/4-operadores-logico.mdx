Operadores Lógicos
====================

Imagine por uma momento pelos menos duas condições para alguém ir a feira.

👵🏻 Tem que ser sábado e ter dinheiro no bolso
-----------------------------------------------

Perfeito. Ou seja, não basta apenas ter dinheiro, nem apenas ser sabado. É preciso que as duas condições sugestões sejam verdadeiras. 


Agora o que precisa ter na feira para você trazer frutas para casa?


👵🏻 Se tiver Maça ou Banana eu trago
---------------------------------------

Legal! Entao basta ter uma das duas frutas para você trazer alguma para casa.

Operadores lógicos servem exatamente para isso. Eles permitem a combinação de condições para uma tomada de decisão mais complexa. 

Operador AND
==============

Este primeiro perador é igual o primeiro caso que vimos.


```javascript
var sexo = "M";
var idade =23;

if (sexo == "M" && idade>=18){
console.log("OK");
}

if (sexo == "M" || idade>=18){
console.log("OK");
}
```
